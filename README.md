# README

This rails application is a scheduling application for Harris Biomedical. This application is intended to be used by a scheduling employee internally within Harris Biomedical, who will schedule new training events as requested by clients. 
The main functionality of the application is to be able to schedule events and add them to a calendar with minimal manual entry required. 

* Ruby version: ruby-2.6.3

* System dependencies:

* Configuration
	The simple_calendar gem is used to render the calendar in this project. Documentation for this gem can be found here: https://github.com/excid3/simple_calendar.
